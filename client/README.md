# OpenLibrary Hub — Client

This is the React (Vite) frontend for OpenLibrary Hub.

Full project documentation — setup instructions for both the frontend and backend, feature list, API reference, and screenshots — lives in the **root README**:

➡️ [../README.md](../README.md)

## Quick start (frontend only)

```bash
npm install
npm run dev
```

Runs at `http://localhost:5173`. Requires the backend (`server/`) to be running separately — see the root README for backend setup.
